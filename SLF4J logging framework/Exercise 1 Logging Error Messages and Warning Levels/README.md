# Exercise 1: Logging Error Messages and Warning Levels (SLF4J)

## Overview

This repository contains the solution for **Exercise 1: Logging Error Messages and Warning Levels** from the **Cognizant Digital Nurture 5.0 – Java FSE React** Deep Skilling Program.

The objective of this exercise is to understand how to use **SLF4J** with **Logback** for logging messages of different severity levels in a Java application.

---

## Objective

* Configure SLF4J in a Maven project.
* Add Logback as the logging implementation.
* Log error and warning messages.
* Understand different logging levels.

## Logging Levels

* TRACE – Detailed debugging information.
* DEBUG – Useful for debugging applications.
* INFO – General application information.
* WARN – Indicates a potential problem.
* ERROR – Indicates a serious problem or failure.

---

## How to Run

1. Create a Maven Project.
2. Add the required SLF4J and Logback dependencies.
3. Create `LoggingExample.java`.
4. Run the application as a Java Application.

---

## Expected Output

```text

WARN  LoggingExample - This is a warning message
```
