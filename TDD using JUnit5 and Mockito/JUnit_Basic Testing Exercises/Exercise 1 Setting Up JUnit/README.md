# Exercise 1: Setting Up JUnit

## Overview

This repository contains the solution for **Exercise 1: Setting Up JUnit** from the **Cognizant Digital Nurture 5.0 – Java FSE React** Deep Skilling Program.

The objective of this exercise is to configure JUnit in a Java project and create a simple unit test to verify the setup.

---

## Objective

* Create a Java project.
* Configure JUnit using Maven.
* Create a simple Java class.
* Create a JUnit test class.
* Execute the test successfully.

---

## Technologies Used

* Java
* JUnit 4.13.2
* Maven
* Eclipse / IntelliJ IDEA

---

## Concepts Covered

* Unit Testing

* JUnit Framework

* Assertions

---

## How to Run

1. Create a Maven Project.
2. Copy the `pom.xml`.
3. Add `Calculator.java`.
4. Add `CalculatorTest.java`.
5. Right-click **CalculatorTest.java**.
6. Select **Run As → JUnit Test**.
