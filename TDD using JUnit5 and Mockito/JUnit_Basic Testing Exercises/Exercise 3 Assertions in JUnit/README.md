# Exercise 3: Assertions in JUnit

This repository contains the solution for **Exercise 3: Assertions in JUnit** from the **Cognizant Digital Nurture 5.0 – Java FSE React** Deep Skilling Program.

## Assertions Used

### assertEquals()

Checks whether the expected value is equal to the actual value.

### assertTrue()

Checks whether the given condition is true.

### assertFalse()

Checks whether the given condition is false.

### assertNull()

Checks whether the object reference is null.

### assertNotNull()

Checks whether the object reference is not null.

---

## How to Run

1. Create a Maven Project.
2. Add the JUnit dependency in `pom.xml`.
3. Create the test class `AssertionsTest.java`.
4. Right-click the test class.
5. Select **Run As → JUnit Test**.
